# 证据等级规则

L0 元数据级：标题、年份、DOI、期刊、作者，只能去重和初筛。
L1 标题摘要级：只能判断候选相关性，不能做数据关系结论、实验流程审计或机制冲突裁决。
L2 正文级：方法、结果、讨论、正文表格，可用于初步变量关系、实验流程和机制主张抽取。
L3 图表级：图、表、图注、曲线、统计图，可用于数据趋势、参数和证据锚点。
L4 补充材料级：补充方法、表格、图和原始参数，可用于深度流程审计和参数库。
L5 原始数据/代码级：用于最高强度复核。

变量关系至少 L2；定量参数最好 L3/L4；实验流程审计至少 L2，常需 L4；数据可比性最好 L2+L3+L4；机制冲突裁决不能只靠 L1。
